## 🛠️ Installation

```shell
git clone https://github.com/chaozwn/yabai.git ~/.config/yabai
```

## Install yabai
```shell
brew install koekeishiya/formulae/yabai
```

## Start yabai
```shell
# start yabai
yabai --start-service
```

## Stop yabai
```shell
# stop yabai
yabai --stop-service
```

## Restart yabai
```shell
# restart yabai
yabai --restart-service
```

## Upgrade yabai
```shell
# upgrade yabai
brew upgrade yabai
```
